How are my messages stored and encrypted?

Your message and key are sent in to the server via SSL/TLS. The server then generates a SHA256 hash of your key, and generates an initilization vector (IV) from the server's /dev/urandom device. Your message is then padded to the correct length using a PKCS7 algorithm. Finally, the server encrypts your message with your key using AES 256 CBC block mode. After encrypting, the IV is prepended to the encrypted message, base64 encoded, and stored in the database. Your key is then discarded. No identifying information about you or your message is stored. All server log messages are sent to /dev/null.

The encryption function is as follows (python): 

def encrypt(self, raw, ukey):
    pad = PKCS7Encoder()                                # Initialize PKCS7 Encoder
    key = hashlib.sha256(ukey).digest()                 # hash key with SHA256
    iv = os.urandom(16)                                 # Get 16 bytes from urandom
    msg = pad.encode(raw)                               # pad the message (% 16 == 0)
    cipher = AES.new(key, AES.MODE_CBC, iv)             # create an encryption object
    return base64.b64encode(iv + cipher.encrypt(msg))   # encrypt, encode, return
